Research

I am interested in all aspects of digital VLSI design. My research focuses on microarchitecture exploration and design using both RTL and high-level synthesis design flows, as well as EDA physical synthesis optimizations. Recent research work includes:

IP for High level synthesis

  • NoCpad: SystemC infrastucture for building scalable industry-grade cache-coherent NoCs
  • Accelerators for sparse/dense clustering algorithms
  • Customized pipelines for Floating point units
  • Graph Neural Networks accelerators

DRiM: DUTH RISC-V Microprocessors

  • DRIM: Low-cost 6-stage pipelined OoO processor dual-issuing only compressed instructions
  • DRIM-S: Two-way superscalar OoO processor
  • RISC-V2: Vector processor for the RISC-V Vector ISA extension

Network on Chip microarchitecture

  • NoCpad: SystemC infrastucture for building scalable industry-grade cache-coherent NoCs
  • Efficient in-network multicasting
  • Low cost clock domain crossing

EDA physical synthesis

  • Incremental timing-driven optimization: placement, sizing, buffering
  • OCV-aware clock tree optimization
  • Machine learning based physical synthesis flow tuning
