Research

My research focuses on Digital VLSI design, Computer architecture, and EDA physical synthesis. Currently my research group designs data-parallel accelerators and RISC-V processors using both RTL and high-level synthesis design flows.

Data parallel accelerators

  • FlashAttention-based parallel hardware accelerators for Transformers and LLMs
  • Accelerators for homomorphic-encryption-based privacy preserving technologies
  • Systolic arrays with optimized reduced-precision floating point arithmetic

DRiM: DUTH RISC-V Microprocessors

  • RISC-V Vector Processors
  • DRIM: Low-cost 6-stage pipelined OoO processor dual-issuing only compressed instructions

IP for HLS design flows

  • FlashAttention-based parallel hardware accelerators for Transformers and LLMs
  • CNN dataflow hardware accelerators (Py2HLS)
  • FastFloat Library of optimized Floating Point hardware operators
  • DRIM4HLS: DUTH RISCV Microprocessor designed in SystemC HLS
  • NoCpad: SystemC HLS infrastucture for building scalable industry-grade cache-coherent NoCs
